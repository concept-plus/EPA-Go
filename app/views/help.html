What is this app?



This application provides users with information about current environmental safety conditions at their location.

Getting Started

To begin, enter a zip code in the search box on the main page and click Go.

Disclaimer

This is a prototype application. This application is solely for the purpose as a proof-of-concept. It uses the EPA EnviroFacts API to retrieve UV light data and the EPA AirNow API to retrieve air quality data.

See www.epa.gov and docs.airnowapi.org for more information.
